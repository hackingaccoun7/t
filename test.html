👋 자기소개

안녕하세요! 저는 웹 프론트엔드 개발자 HA7입니다. 사용자 경험(UX)과 애플리케이션 성능을 최우선으로 고려하여, 최적화된 웹 솔루션을 설계하고 구현하는 것을 즐깁니다. 새로운 기술과 도구들을 탐험하며, 그것들을 실무에 적절히 적용해 가는 과정에서 끊임없이 성장하고 있습니다. 효율적인 코드 작성과 유연한 설계를 통해, 사용자에게 최상의 경험을 제공하는 웹 환경을 만드는 것을 목표로 합니다.

🛠️ 기술 스택

  • HTML5
  • CSS3 / Tailwind
  • JavaScript (ES6+)
  • React
  • Vue.js
  • Node.js
  • Git & GitHub
  • Figma
  • 네트워크 보안
  • 침투 테스트 (Penetration Testing)
  • 암호화 및 해독 기술
  • OWASP Top 10
